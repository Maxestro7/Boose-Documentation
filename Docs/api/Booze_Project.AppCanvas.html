Class AppCanvas

Concrete application implementation of the BOOSE.ICanvas interface. Provides simple drawing primitives backed by an in-memory Bitmap and a Graphics surface. The canvas maintains a current pen position (Xpos, Ypos) used as the origin for many drawing operations.

Gets or sets the pen colour used for drawing. The property type is object to match the BOOSE interface:

  • Setter accepts a Color or a string (HTML hex like "#RRGGBB"); invalid or unsupported values are ignored.
