Class AppCommand

Base application command implementing IAppCommand. Provides common parameter handling, a reference to the containing AppStoredProgram, and basic compile/execute lifecycle behaviour used by concrete command implementations.

Remarks

Concrete commands should derive from AppCommand and override checkparameters(string[]) and Execute() where appropriate. The Compile() method processes the textual parameter list and validates parameters prior to runtime execution.
