Class AppCanvasCommand

Base class for application commands that operate on a drawing canvas. Inherits from AppCommand and exposes a protected BOOSE.ICanvas instance plus pen position fields used by concrete canvas commands (for example: rect, circle, moveto).

Syntax
internal class AppCanvasCommand : AppCommand, IAppCommand
Remarks

Concrete canvas commands should derive from AppCanvasCommand to gain access to the shared canvas instance and the inherited parameter handling from AppCommand. The protected xPos and yPos fields represent the current pen position and are typically used or updated by drawing commands.
