Class AppCircle

Represents the circle command. Validates a single integer parameter (radius) and instructs the canvas to draw a circle. Inherits from AppCommand1Param, the application-level single-parameter command base which derives from AppCanvasCommand.

Execute()

Executes the circle command:

  • Calls base.Execute() to perform any base-class processing.
  • Ensures the provided parameter is an integer (not a double).
  • Reads the integer radius from the inherited paramsint.
  • Validates the radius is >= 1.
  • Requests the canvas to draw the circle via canvas.Circle(radius, false).
