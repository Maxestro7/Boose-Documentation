Class AppCommand2Param

Base class for application commands that accept exactly two parameters. Derives from AppCommand1Param and enforces a two-parameter count during compilation.

Remarks

Concrete two-parameter commands (for example rect) should derive from AppCommand2Param. The class provides a default parameter-count validation by overriding checkparameters(string[]).
