Class AppCommandFactory

Singleton factory that creates application-specific IAppCommand instances. Maps textual command names to concrete App* implementations used by the application.

Remarks

Use Instance to obtain the singleton factory. Unknown command names result in a BOOSE.FactoryException. The factory uses a simple double-checked lock to ensure thread-safe lazy initialization.
