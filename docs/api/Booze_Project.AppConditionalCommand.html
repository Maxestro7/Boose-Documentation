Class AppConditionalCommand

Base class for commands that represent conditional behaviour (blocks/expressions). Inherits boolean evaluation behavior from AppBool and adds line-number bookkeeping and a runtime condition used to control program flow.

public class AppConditionalCommand : AppBool, IAppCommand
Remarks

Instances hold their source line number (Linenum), the index to jump to when the condition is false (EndLineNum), and the evaluated boolean result (Condition). The program/compilation phase is expected to set EndLineNum and Linenum.

Remarks

Calls base.Execute() to evaluate the boolean expression via the AppBool/ AppEvaluation logic. The evaluated boolean is copied into Condition; if false the program's PC is set to EndLineNum.
