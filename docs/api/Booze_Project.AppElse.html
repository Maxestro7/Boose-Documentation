Class AppElse

Represents an application-level else compound command. Acts as the partner for a preceding conditional (if) and links to the subsequent end command.

Remarks

During compilation the AppElse pops the matching conditional command from the program condition stack, records linking information (line numbers) and pushes itself so that the following end command can complete the pairing. At runtime the Execute() implementation advances the program counter to the end of the else block.
