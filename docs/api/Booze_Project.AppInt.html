Remarks

During Compile() the variable declaration/expression is parsed by the base class. At runtime Execute() evaluates the expression (via the program evaluator), enforces integer semantics (rejecting real-to-int conversions) and writes the resulting integer value back to the program's variable store using program.UpdateVariable.

Execute()

Runtime evaluation and assignment for an integer variable.

  • Calls Execute() to evaluate the stored expression into evaluatedexpression
  • Attempts to parse the evaluated expression as an int.
  • If parsing fails but the value is a real number, throws a BOOSE.StoredProgramException indicating a real-to-int conversion is invalid.
  • If parsing fails for any other reason, throws a BOOSE.StoredProgramException indicating an invalid integer expression.
  • On success updates the program variable via program.UpdateVariable(this.VarName, result).
