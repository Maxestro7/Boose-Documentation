Remarks

The command expects three integer parameters supplied via the inherited comma-separated parameter list. Parameters are resolved through paramhandler() which will look up variables or parse numeric literals. On execution the RGB components are passed to the underlying canvas using canvas.SetColour(r, g, b).

  • Calls base.canvas.SetColour(r, g, b) to update the canvas pen colour.
