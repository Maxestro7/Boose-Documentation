Class AppReal

Application implementation for real (double) variable evaluation/assignment. Inherits parsing and evaluation behaviour from AppEvaluation and stores the evaluated value as a double.

Remarks
  • During Compile(), the base class parses the parameter list and this instance is registered with the containing program variable table.
  • At runtime, Execute() evaluates the expression (via the program evaluator) and converts the result to double.
  • The class intentionally hides the integer Value from AppEvaluation with a new double-typed property to represent real values.
