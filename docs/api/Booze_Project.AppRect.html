Remarks

Inherits from AppCommand2Param, the application-level two-parameter command base. Parameters are resolved via paramhandler() which looks up variables or parses numeric literals. On successful validation the command calls canvas.Rect(width, height, false).

Execute()

Executes the rect command:

  • Calls base.Execute() to perform any base-class processing.
  • Resolves parameters to integers via paramhandler.
  • Validates dimensions are at least 1.
  • Calls canvas.Rect(width, height, false) to draw the rectangle.
