Class Appif

Application implementation of the if compound command. Evaluates the conditional expression and, if the condition is false, advances the program counter to the command indexed by EndLineNum.

Remarks

Appif derives from AppCompoundCommand and relies on the compilation phase to set up the associated conditional linking (e.g. CorrespondingCommand and EndLineNum). At runtime the expression is normalised using expressionhandler() and evaluated via Program.EvaluateExpression. The method expects the evaluated expression to return a textual boolean ("True"/"False") that can be parsed by Parse(string)

Execute()

Execute the if behaviour:

  • Normalises the conditional expression using expressionhandler().
  • Evaluates the expression via Program.EvaluateExpression(string).
  • If the resulting boolean is false, sets program.PC to EndLineNum to skip the if-block.
