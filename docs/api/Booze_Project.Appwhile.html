Class Appwhile

Application implementation of the while compound command. Evaluates a boolean expression each iteration and, when false, advances the program counter to the linked end command.

Remarks

Appwhile derives from AppCompoundCommand and relies on the compilation phase to push the conditional onto the program stack so it can later be linked with an end command. The Compile() method records the index of the first command in the loop so the end command can jump back when needed. At runtime Execute() normalises the conditional expression using expressionhandler() and evaluates it via program.EvaluateExpression(string); a false result causes execution to continue after the loop.

Execute()

Runtime execution for the while loop:

  • Normalises the loop condition using expressionhandler().
  • Evaluates the condition using EvaluateExpression(string).
  • If the evaluated boolean is false, sets EndLineNum target by assigning program.PC = EndLineNum so execution continues after the loop.

Remarks

The implementation evaluates the condition once per Execute() call. When the condition is true the method exits normally and execution continues with the next command; when false the program counter is set to the loop's end. The method expects the expression evaluator to return a boolean string parseable by Parse(string).
