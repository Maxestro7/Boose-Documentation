Interface IAppParser

Defines the contract for a parser that converts textual program source into application command instances.

Namespace: Booze_Project
Assembly: Booze Project.dll
Syntax
public interface IAppParser
Remarks

Implementations should:

  • Tokenise and normalise source lines,
  • Create concrete IAppCommand instances using an IAppCommandFactory,
  • Assign commands to an AppStoredProgram and call Compile() for each command,
  • Provide both single-line parsing and multi-line program parsing entry points.

Remarks
