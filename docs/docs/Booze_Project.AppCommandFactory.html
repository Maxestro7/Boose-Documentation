Remarks

This factory maps textual command names produced by the parser to concrete command implementations used by the application. Currently supported command names are:

  • "circle" → AppCircle
  • "moveto" → AppMoveTo
  • "rect" → AppRect
If the command name is not recognised here the call is forwarded to the base factory implementation.
