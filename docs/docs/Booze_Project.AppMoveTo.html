Application implementation of the BOOSE.CommandTwoParameters for the moveto command. Reads two integer parameters (x, y) from the inherited paramsint array and instructs the associated canvas to move the pen position to that location.

Remarks

The command expects integer parameters. The command validates that the supplied coordinates are within the drawing bounds (0..600). If the coordinates are out of range a BOOSE.BOOSEException is thrown.

Remarks

Behaviour:

  • Calls base.Execute() to allow base class processing.
  • Reads integer parameters from the inherited paramsint array: x = paramsint[0] and y = paramsint[1].
  • Validates the coordinates are within the allowed drawing box (0..600).
  • If validation passes, calls Canvas.MoveTo(x, y) to update the pen position.

Exceptions

BOOSEException

Thrown when the supplied coordinates are outside the drawing bounds.
