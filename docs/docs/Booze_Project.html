Namespace Booze_Project

Classes

AppCircle

Represents the 'circle' command. Validates a single integer parameter and instructs the canvas to draw a circle of the specified radius. Inherits from BOOSE.CommandOneParameter.

AppCommandFactory

Application-specific command factory that creates concrete BOOSE.ICommand instances for commands supported by the application and delegates unknown commands to the base BOOSE.CommandFactory.

AppMoveTo

Application implementation of the BOOSE.CommandTwoParameters for the moveto command. Reads two integer parameters (x, y) from the inherited paramsint array and instructs the associated canvas to move the pen position to that location.

AppRect

Application implementation of the BOOSE.CommandTwoParameters for the rect command. Validates two integer parameters (width and height) and instructs the canvas to draw a rectangle.

ApplicationConfiguration

Bootstrap the application configuration.

Booze_UI

Main Windows Forms UI for the BOOSE application. Hosts the drawing canvas, the command factory, parser and stored program, and exposes event handlers for user interaction (run, reset, paint, etc.).

IAPPCanvas

Application implementation of the BOOSE.ICanvas interface. Provides drawing primitives backed by a Bitmap and a Graphics object. The canvas stores a pen position (Xpos, Ypos) which many commands use as the current drawing origin.
